stuQuery

I often use jQuery but it was becoming too big for most of my use cases. I only needed a few basic features. So I've gone back to basics and made my own small library. The aim was to make the usage be as similar to jQuery as possible so that I can switch back-and-forth if I need to. To avoid conflicts, rather than $ I've used E.

Basic example

E(document).ready(function(){
	// Things that happen once the document is loaded

});

File loading

function success(data,a){
	E('#output').append('<p>Success for <em>'+a.url+'<\/em><\/p>');
	E('#output').append("<textarea>"+data+"<\/textarea>");
}
function error(e,a){
	E('#output').append('Failed to load '+a.url);
}
E().ajax("file.txt",{'complete': success, 'error': error });	// This file exists
E().ajax("nofile.txt",{'complete': success, 'error': error });	// This file doesn't exist
